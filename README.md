# 🧠 NLP Streamlit App

Функции:
- Анализ тональности
- Генерация текста
- Суммаризация

Поддержка:
- Русский
- English
- Автоопределение языка

---

## 🚀 Установка

pip install -r requirements.txt

---

## ▶️ Запуск

streamlit run app.py

---

## 🌐 Открыть

http://localhost:8501

---

## ⚠️ Важно

- Первый запуск долгий (скачиваются модели)
- Далее работает быстрее (кеш)
